This document contains the full API documentation of the Fuego Core Middleware Service Set II. Included components are the Message Transport Service and Protocol, the Rendezvous-Notify event service, the Syxaw synchronizing file system, the mobile presence service, and the Fuego-SIP event gateway. In addition, a utility package of general-purpose classes is provided.

The precise specification for these services as well as an overview of their implementation are provided in Fuego Core deliverables.
