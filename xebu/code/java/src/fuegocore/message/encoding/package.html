Serializers and parsers for the Xebu format. This package contains everything needed to use the Xebu format for XML data in programs.

The serializer and parser are implementations of the XmlPull APIs {@link org.xmlpull.v1.XmlSerializer} and {@link org.xmlpull.v1.XmlPullParser}, respectively, so these classes can be used as drop-in replacements in existing applications.
