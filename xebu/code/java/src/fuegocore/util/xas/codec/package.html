Typed content encoders and decoders for common Java types. This package contains classes implementing the {@link fuegocore.util.xas.ContentEncoder} and {@link fuegocore.util.xas.ContentDecoder} requirements for some common types. Types specific to the Fuego Core system components are handled in packages relevant to those components.

This package also contains the {@link fuegocore.util.xas.codec.DefaultCodecFactory} class to be used as a default {@link fuegocore.util.xas.ContentCodecFactory}. It recognizes some useful Java types.
